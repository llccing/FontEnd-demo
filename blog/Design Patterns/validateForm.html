优点：避免了全局变量，一旦出现覆盖现象，会全部覆盖。

问题：当别人想用你的方法时，不能复制一份，或者说别人不能通过new的方式继承这些方法。

# 使用返回对象的形式demo4

    var a = checkObject();
    a.checkPassword();

# 使用类的形式 demo5

    // 此处要通过new关键字来创建对象，因为使用的是类的形式。会改变this的指向到a
    var a = new checkObject();
    a.checkPassword();

写成类的形式之后，每个人在new之后都会有独有一套方法，但是这样做也会带来性能问题，需要改变一下。
