# 数据驱动

Vue.js的核心思想是数据驱动。
视图是由数据驱动生成的，我们对视图的修改，不会直接操作DOM，而是通过修改数据。

# 好处

  • 简化代码量
  • 利于维护

简洁的模板语法来声明式的将数据渲染为DOM：

<div id="app">
  {{message}}
</div>

var app = new Vue({
  el: '#app',
  data: {
    message: 'Hello Vue!',
  }
})
