# React

最开始接触前端框架时，第一个听说的就是React，那时还不知道什么是Vue，React像是神一样的存在。当然最开始时也尝试过React搭建项目，可能复杂度比较高，所以最后不了了之了。

那么，在用过Vue近两年的时间里，心里始终有个结，React是个坎，是一个一定要过去的坎。

最好的学习就是实践！
