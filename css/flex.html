# 认识Flex

2009年，W3C提出了一种新的解决方案----Flex布局，可以简便、完整、响应式的实现各种页面布局。

一、Flex 是 Flexible Box缩写， “弹性布局”，用来为盒状模型提供最大的灵活性。 任何一个容器都可以指定Flex布局。
